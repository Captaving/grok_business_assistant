# Grok Business Assistant Bot

Персональный Grok-ассистент, который работает в личных чатах пользователей через **Telegram Business**.

## Возможности

- Пользователь пишет свой промт (до 300 символов)
- Бот сохраняет промт, язык, длину истории и задержку ответа
- Полноценный тестовый режим перед подключением
- Умные ответы через **Grok API** с контекстом 6–12 сообщений
- Распознавание голосовых (Groq Whisper) — легко добавить
- Работает в **личных чатах** клиента через Telegram Business
- Задержка ответа (выглядит естественно)

## Установка

1. Клонируй репозиторий
2. `cp .env.example .env` и заполни ключи
3. `pip install -r requirements.txt`
4. Включи **Business mode** у бота в @BotFather
5. `python bot.py`

## Важно

- Бот должен быть подключён пользователем через **Настройки → Telegram для бизнеса → Чат-боты**
- Для работы в бизнес-режиме у пользователя должен быть Telegram Premium (на момент 2026 года)

## Что дальше можно добавить

- Полноценная транскрипция голосовых в business_message
- Поддержка изображений
- Админ-панель со статистикой

- Несколько промтов на одного пользователя (разные сценарии)
